# PATH for Android

When working with Cordova and running commands from the Terminal to create Android apps we are going to need to update our PATH variable and create a few more ENV variables.

We will need to add these updates to our .zshrc file.

export STUDIO_JDK=$(/usr/libexec/java_home)
export ANDROID_SDK_ROOT="/Users/<username>/Library/Android/sdk"
export GRADLE_HOME="/Applications/Android Studio.app/Contents/gradle/gradle-6.0.1"
